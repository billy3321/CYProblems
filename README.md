# 監察委員被提名人爭議彙整

## 本書特色

這是由沃草[國會無雙](http://musou.tw/)編輯群所整理的「監察委員被提名人爭議彙整」。

在本書中，我們把每個被提名人的摘要文字、新聞連結都附上，方便大家簡單的查看、查找。

我們希望這樣的書本，可以凸顯出監察委員被提名人的問題，也可凸顯出台灣黨政不分的荒謬現象。
